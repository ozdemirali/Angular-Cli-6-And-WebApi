# Angular-Cli-6-And-WebApi

Content:
	There are two projects
* ShopTemplateCli
* ShopTemplateWebApi


ShopTemplateCli:
* This project is created whit AngularCli ^6.1.0.
* Nodejs is neded.
* ng serve –o(This Project is run http://localhost:4200/).


ShopTemplateWebApi:
* This Project is created whit WebApi template 4.6.1(Restful http Service) in Asp.Net Web Aplication(.Net Framework).
* EntityFramework v6.1.3 is user for database.
* Localhost: http://localhost:52378/

ASP.NET Web API Help Page
